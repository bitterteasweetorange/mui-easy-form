# CascaderIO

CascaderIO let users select a value from a set of assocaiaed values.

## Features

- filter nodes by keyword
- highlight the keyword

<Canvas of={Stories.Default} />

- if `typeof value === 'string'` or `typeof value === 'number'`, value will be used as label
- if value's type is `{label:string}`, label will be used as label
- otherw, you shoule privide a `getOptionLabel` prop to decide label's display
